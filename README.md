# HPC-Talk
Seminar presentation and code on how HPC thinking helps real-word problems

Slides are [here](https://github.com/hal-elrod/HPC-Talk/blob/main/Baylor%20High%20Performance%20Presentation.pdf)

* [matrices](matrices/) has multiple implementations of NxN matrix multiplication for comparison of methods and performance
* [queue](queue) has simple queue simulator (SimPy) with uniform or logrithmic arrival times, uniform or lognormal handling times
